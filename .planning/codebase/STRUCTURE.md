## Naming Conventions

**Files:**

- Primary script: `Universal Spell Checker.ahk` (no version suffix on active file)
- Legacy variants: `Universal Spell Checker - DESCRIPTION.ahk` or in `Old Spell Check Version Files/`
- Configuration: lowercase with underscores (`replacements.json`, `spellcheck.jsonl`)
- Generated: lowercase with underscores (`viewer.html`, `spellcheck-TIMESTAMP.log`)
- Documentation: UPPERCASE.md for GSD docs (`ARCHITECTURE.md`, `CONVENTIONS.md`)

**Directories:**

- Hidden metadata: Dot prefix (`.planning`, `.git`, `.github`, `.claude`)
- Log data: `logs/` (lowercase, plural)
- Archive: `Old Spell Check Version Files/` (descriptive, with spaces)

**Code Entities:**

- Functions: PascalCase for exported/user-visible (`LoadReplacements()`, `ApplyReplacements()`)
- Private functions: `__DoubleUnderscore` prefix for internal helpers (`__JsonParseObject()`, `__ExtractHtmlFragment()`)
- Global variables: camelCase (`modelModule`, `apiModel`, `postReplacements`)
- Constants: SCREAMING_SNAKE_CASE (Windows API constants: `CF_TEXT`, `GMEM_MOVEABLE`)
- Local variables: camelCase (`originalText`, `correctedText`, `logData`)

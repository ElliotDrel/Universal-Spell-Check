Universal Spell Checker App Folder
=====================================

This folder contains the C# .NET Windows Forms application version of the Universal Spell Checker.

IMPORTANT: This folder is not committed to git for storage reasons.

Contents:
- Complete C# .NET 8.0 Windows Forms application
- Visual Studio project files
- Built executables and dependencies
- Inno Setup installer configuration
- Debug and release builds

If you need access to the C# application, it's available on Elliot's laptop (school).

Note: The main project focus is on the AutoHotkey script (Universal Spell Checker.ahk) 
in the root directory, which is the actively developed and optimized version.
